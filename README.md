The project for "Marlo" Hackaton 2025 made by QoralTeam
## About
A telegram bot for It.Moscow college students that sends notifications of the upcoming classes and schedules.  

## Required software
- Python 3.11+ ([Download](https://www.python.org/downloads/))
- Git 2.x+ ([Download](https://git-scm.com/downloads))

## Get started 
### Downloading Python (3.11+)  
#### 1) Download the installer:  
Windows / macOS / Linux: https://www.python.org/downloads/  

#### 2) Run the installer:
- Windows:   
Double-click the downloaded .exe. Check “Add Python to PATH” then  
click Install. 
- macOS: open the .pkg and follow prompts. You can also use Homebrew: brew install python@3.11.
- Linux: use your distro package manager or download from python.org. Example (Ubuntu):

    
    bash
    sudo apt update
    sudo apt install -y software-properties-common
    sudo add-apt-repository ppa:deadsnakes/ppa
    sudo apt update
    sudo apt install -y python3.11 python3.11-venv python3.11-dev
Verify installation:

    bash
    python3.11 --version
    pip install --upgrade pip
